# 🚀 Portfolio 2026 | Interactive Experience

Concepto de portfolio interactivo de alto rendimiento utilizando tecnologías de última generación.

## 🛠 Tech Stack
* **Framework:** Next.js 15 (App Router)
* **Language:** TypeScript
* **3D Engine:** React Three Fiber (Three.js)
* **Styling:** Tailwind CSS
* **Animations:** GSAP / Framer Motion

## 🏗 Estructura del Proyecto
- `/src/app`: Rutas y vistas principales.
- `/src/components/canvas`: Escenas 3D e interactividad.
- `/src/components/ui`: Componentes de interfaz futurista.
